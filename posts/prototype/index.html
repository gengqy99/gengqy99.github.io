构造函数的 __proto__ 是什么？

在上图中可以看到，构造函数对象 Person 的构造函数是 Function。

不难想到，我们可以通过 new Function() 的方式定义一个函数，语法如下：

var fun = new Function('arg1', 'arg2', `
  arg1 = "参数1";
  arg2 = "参数2";
  console.log("别的执行语句");
  return 0;
`);

尽管这样是可行的，但不建议这样定义函数。因为这会解析两次代码，第一次解析常规js代码，第二次解析传入构造函数的字符串，效率低
